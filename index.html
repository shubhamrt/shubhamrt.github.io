Vision

Rangelands and drylands hold soil, water, food security, and culture. When connectivity breaks — vegetation fragments, bare patches link up, runoff accelerates — systems can shift fast and become hard to reverse.

My work aims to identify where those breaks start, when they're getting worse, and how to intervene early.

Research Mission

I study how drylands break apart — and how we can stitch them back together.

My work links patch-scale processes (bare soil, runoff channels, shrub islands) to landscape-scale risk: sediment loss, grazing collapse, and long-term ecosystem resilience.

I'm an early-career researcher and hydrologist working on land degradation, erosion pathways, and eco-geomorphic resilience in drylands. I use network theory to map how water, sediment, and vegetation are connected. Those maps then help grazing managers decide where to rest paddocks, re-seed, or add barriers — before the land shifts into a degraded state.

When small bare spots start linking up

Bare soil between shrubs creates tiny flow paths. Those paths join, speed up runoff, and pull sediment downslope. I map those paths as a network: each patch is a node, each flow path is an edge. The goal is to find leverage points where a tiny, local fix protects the whole slope.

  • Grass → shrub regime shifts
  • Aridity + herd pressure
  • Wind-driven sediment feedbacks
